## 🚀 部署步骤（10分钟上线）

### 步骤1: 上传到GitHub（3分钟）
1. 访问 github.com，创建新仓库
2. 仓库名称：`ai-tools-hub`
3. 设置为Public（Vercel免费托管需要）
4. 上传`AI_Tools_International`文件夹中的所有文件

### 步骤2: 部署到Vercel（5分钟）
1. 访问 vercel.com，用GitHub账号登录
2. 点击"Import Project"
3. 选择你的`ai-tools-hub`仓库
4. Framework选择"Other"（静态HTML）
5. 点击"Deploy"
6. 等待1-2分钟，完成！

### 步骤3: 获取网站链接（1分钟）
部署完成后，Vercel会给你一个链接：
```
https://ai-tools-hub-xxx.vercel.app
```

**立即测试:**
- 打开链接
- 检查首页是否正常显示
- 点击工具卡片
